2- Journey to Istanbul

It is full-stack CRUD application about visiting blog. I did it to tell about Istanbul, the city where I was born. In visitor mode you can just see the places and read information about that.If you are logined user, you can create and delete historical or sightseeing place about Istanbul. You can add restaurants for this places and give point for restaurants. Also you can make comments for this places and update or delete these comments.

In this project i used HTML5, CSS and Javascript for front-end. In back-end side i used Express, Mongoose and MongoDB. You can see my app in here.
